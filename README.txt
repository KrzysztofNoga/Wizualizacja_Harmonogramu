# Wizualizacja harmonogramów

Projekt stworzono w celu zwizualizowania harmonogramów zajęć. Pobiera dane 
z pliku JSON przekazuje do odpowiednich elementów blokowych, gdzie przy pomocy
kaskadowych arkuszów stylów zostają formatowane.

## Instalacja

1.Pobrać pliki źródłowe
2.Następnie przeslać na serwer
3.Uruchomic w przeglądarce
 

## Zadania

Zadaniem projektu jest wizualizacja i porównywanie harmonogramów.
Należy wybrać co najmniej jedną grupę i tydzień i nacisnąć OK.
Wyświetlony zostanie harmonogram.
